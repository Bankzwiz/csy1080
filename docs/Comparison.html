  • Web developer

    A Web Developer specializes in creating websites or they handle improving existing websites. They carry out such jobs using specialist programming code such as HTML5, and JavaScript. Web Developers provide a comprehensive service to business who are looking to create an attractive website which is appealing to their target customers. The main responsibility of a Web Developer is for launching a website, or improving its design and maintenance of a website.

    Salary: £24,841 per year
